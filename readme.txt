This plugin will add a nice scroll to top button to your website. It has a nice option panel. you can change button's color button's icon.

== Description ==

AS Scroll To Top is a plugin that let you add a nice Scroll To Top button to your WordPress blog. 
<br>
Here Is the Demo of it: http://ashik.freeserver.me/as-scroll-to-top/
<br>
<br>

<strong> Features: </strong><br>
1. Just plug and play.<br>
2. Use 100's of icons to your button.<br>
3. change color of your button with css.<br>
4. change size of your button with css. (For sizing your button use font size and line height css property)<br>
5. Just write Your Icon name to Change icon of your button. You can Get icons name <a href="http://fontawesome.io/icons/#directional">Here</a><br>

== Installation ==

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
